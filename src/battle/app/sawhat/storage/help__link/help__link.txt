Classic markdown [link](http://www.example.com) are made like this: `[link text](http://www.example.com)`.  
You can add a [title at your link](http://www.example.com &quot;Example title&quot;) by adding it right after the url like this: `[link text](http://www.example.com &quot;Link title&quot;)`.  
You can also directly type an url to create an automatic link http://example.com.
